Análise Comparativa entre ShellSort e HeapSort

Questão 1:
O HeapSort possui complexidade O(n log n) garantida porque realiza operações de heap com custo logarítmico para cada elemento. Já o ShellSort depende da sequência de gaps, o que torna sua complexidade variável.

Questão 2:
